üì¶ Core Components

Foundation classes for parsing, building, and manipulating JSON

SIMPLE_JSON

SIMPLE_JSON

Main entry point for all JSON operations. Provides parsing, building, and error tracking capabilities.

  • Parse JSON from strings or files
  • Build JSON with fluent API
  • Comprehensive error tracking
  • Line/column position information
  • Full Unicode/UTF-8 support

SIMPLE_JSON_VALUE

SIMPLE_JSON_VALUE

Base wrapper for all JSON values with type checking and safe conversion methods.

  • Type checking (is_string, is_number, etc.)
  • Safe type conversion (as_string_32, as_integer, etc.)
  • STRING_32 for all text values
  • Access to underlying JSON_VALUE

SIMPLE_JSON_OBJECT

SIMPLE_JSON_OBJECT

JSON object (map/dictionary) with fluent API for building and manipulation.

  • put_* methods for all types
  • Fluent API with method chaining
  • Key existence checking
  • Type-safe item retrieval
  • Pretty printing support

SIMPLE_JSON_ARRAY

SIMPLE_JSON_ARRAY

JSON array (list) with fluent API for building and manipulation.

  • add_* methods for all types
  • Fluent API with method chaining
  • Index-based access
  • Count and iteration support
  • Pretty printing support

SIMPLE_JSON_ERROR

SIMPLE_JSON_ERROR

Error information with line/column position tracking for debugging.

  • Error message
  • Line and column numbers
  • Source line context
  • Position indicator
  • Formatted output

üß≠ Navigation & Querying

RFC-compliant path navigation and query capabilities

JSON Pointer RFC 6901

SIMPLE_JSON_POINTER

Navigate JSON documents using standard pointer syntax: /path/to/value

  • RFC 6901 compliant
  • Path-based navigation
  • Array index support
  • Escaped character handling
  • Parent path navigation

JSONPath Queries

via SIMPLE_JSON_OBJECT/ARRAY

Query JSON using path patterns with wildcards and array selectors.

  • Dot notation ($.users.name)
  • Array wildcards ($.users[*])
  • Multi-level queries
  • Result collection

üîß Document Modification

RFC-compliant operations for modifying JSON documents

JSON Patch RFC 6902

SIMPLE_JSON_PATCH

Apply structured modification operations to JSON documents.

  • Add operation
  • Remove operation
  • Replace operation
  • Move operation
  • Copy operation
  • Test operation

JSON Merge Patch RFC 7386

SIMPLE_JSON_MERGE_PATCH

Simple merge semantics for updating JSON with partial modifications.

  • Recursive merging
  • Null = delete semantics
  • Simpler than JSON Patch
  • RFC 7386 compliant

JSON Schema Draft 7

SIMPLE_JSON_SCHEMA

Validate JSON documents against JSON Schema specifications.

  • Type validation
  • Property constraints

SIMPLE_JSON_SCHEMA_VALIDATOR

Internal validator engine with comprehensive error reporting.

  • Detailed error messages
  • Path-based error reporting

  • Recursive validation
