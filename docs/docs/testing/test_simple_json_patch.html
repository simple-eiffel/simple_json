Overview

TEST_SIMPLE_JSON_PATCH validates the implementation of RFC 6902 (JSON Patch), a standard for expressing a sequence of operations to apply to a JSON document. JSON Patch provides fine-grained control over document transformations using six operation types.

JSON Patch Operations

Operation Description Parameters
add Add value at path path, value
remove Remove value at path path
replace Replace value at path path, value
move Move value from one path to another from, path
copy Copy value from one path to another from, path
test Assert value at path equals expected path, value

Test Categories

1. Add Operation Tests

test_add_object_member
Add new property to object
Document: {"foo": "bar"}
Patch:    [{"op": "add", "path": "/baz", "value": "qux"}]
Result:   {"foo": "bar", "baz": "qux"}
test_add_array_element
Add element to array at specific index
Document: [1, 2, 3]
Patch:    [{"op": "add", "path": "/1", "value": 99}]
Result:   [1, 99, 2, 3]
test_add_array_end
Add element to end of array using "-"
Document: [1, 2, 3]
Patch:    [{"op": "add", "path": "/-", "value": 4}]
Result:   [1, 2, 3, 4]

2. Remove Operation Tests

test_remove_object_member
Remove property from object
Document: {"foo": "bar", "baz": "qux"}
Patch:    [{"op": "remove", "path": "/baz"}]
Result:   {"foo": "bar"}
test_remove_array_element
Remove element from array
Document: [1, 2, 3, 4]
Patch:    [{"op": "remove", "path": "/2"}]
Result:   [1, 2, 4]

3. Replace Operation Tests

test_replace_value
Replace value at path
Document: {"foo": "bar"}
Patch:    [{"op": "replace", "path": "/foo", "value": "baz"}]
Result:   {"foo": "baz"}
test_replace_object_with_array
Replace entire object with different type
Document: {"foo": {"nested": "object"}}
Patch:    [{"op": "replace", "path": "/foo", "value": [1, 2, 3]}]
Result:   {"foo": [1, 2, 3]}

4. Move Operation Tests

test_move_object_member
Move property from one key to another
Document: {"foo": "bar", "baz": "qux"}
Patch:    [{"op": "move", "from": "/foo", "path": "/moved"}]
Result:   {"baz": "qux", "moved": "bar"}
test_move_array_element
Move array element to different position
Document: [1, 2, 3, 4]
Patch:    [{"op": "move", "from": "/0", "path": "/3"}]
Result:   [2, 3, 4, 1]

5. Copy Operation Tests

test_copy_value
Copy value from one path to another
Document: {"foo": "bar"}
Patch:    [{"op": "copy", "from": "/foo", "path": "/baz"}]
Result:   {"foo": "bar", "baz": "bar"}
test_copy_nested_object
Deep copy of nested structure
Document: {"original": {"nested": {"deep": "value"}}}
Patch:    [{"op": "copy", "from": "/original", "path": "/copy"}]
Result:   {"original": {"nested": {"deep": "value"}}, 
          "copy": {"nested": {"deep": "value"}}}

6. Test Operation Tests

test_test_operation_success
Test operation passes when values match
Document: {"foo": "bar"}
Patch:    [{"op": "test", "path": "/foo", "value": "bar"}]
Result:   No changes (test passes)
test_test_operation_failure
Test operation fails when values don't match
Document: {"foo": "bar"}
Patch:    [{"op": "test", "path": "/foo", "value": "baz"}]
Result:   Error: test failed

7. Complex Patch Tests

test_multiple_operations
Apply sequence of operations
Document: {"foo": "bar"}
Patch:    [
    {"op": "add", "path": "/baz", "value": "qux"},
    {"op": "replace", "path": "/foo", "value": "updated"},
    {"op": "remove", "path": "/baz"}
]
Result:   {"foo": "updated"}
test_nested_path_operations
Operations on nested structures
Document: {"a": {"b": {"c": "value"}}}
Patch:    [{"op": "replace", "path": "/a/b/c", "value": "new"}]
Result:   {"a": {"b": {"c": "new"}}}
